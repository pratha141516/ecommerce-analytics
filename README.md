# E-Commerce Analytics Project

## Project Overview

This project simulates a real-world E-Commerce Analytics system using PostgreSQL, Docker, and SQL.

The objective is to analyze customer purchasing behavior, product performance, revenue trends, and business insights using advanced SQL queries.

---

## Analytics Implemented

### Customer Spending Analysis

Identified top customers based on total spending.

### Revenue by Category

Calculated revenue generated by each product category.

### Customer Ranking

Used SQL Window Functions (RANK()) to rank customers.

### High Value Customers

Used CTEs to identify customers spending more than ₹1000.

### Average Order Value

Calculated average order value across all orders.

### Monthly Revenue Analysis

Tracked revenue trends month-wise using DATE_TRUNC().

## Docker Setup

Start PostgreSQL Container:

```bash
docker compose up -d
```

Access Database:

```bash
docker exec -it ecommerce-postgres psql -U postgres -d ecommerce
```
