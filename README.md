# E-Commerce Analytics Project

## Overview

A Dockerized PostgreSQL-based E-Commerce Analytics project designed to analyze customer purchasing behavior, product performance, and revenue trends using SQL.

This project demonstrates database design, SQL analytics, Docker containerization, and Git/GitHub workflow.

---

## Tech Stack

- PostgreSQL 16
- Docker
- Docker Compose
- SQL
- Git
- GitHub

---

## Database Schema

### Customers
Stores customer information.

| Column | Type |
|----------|----------|
| customer_id | SERIAL PRIMARY KEY |
| customer_name | VARCHAR(100) |
| email | VARCHAR(100) |
| city | VARCHAR(50) |
| signup_date | DATE |

### Products

| Column | Type |
|----------|----------|
| product_id | SERIAL PRIMARY KEY |
| product_name | VARCHAR(100) |
| category | VARCHAR(50) |
| price | DECIMAL(10,2) |

### Orders

| Column | Type |
|----------|----------|
| order_id | SERIAL PRIMARY KEY |
| customer_id | INT |
| order_date | DATE |
| total_amount | DECIMAL(10,2) |

### Order Items

| Column | Type |
|----------|----------|
| order_item_id | SERIAL PRIMARY KEY |
| order_id | INT |
| product_id | INT |
| quantity | INT |

## Analytics Implemented

### Revenue by Category

Analyzes revenue generated by each product category.

### Top Customers

Ranks customers based on total spending.

### Average Order Value

Calculates average revenue per order.

### Monthly Revenue

Tracks revenue trends over time.

### Customer Ranking

Implemented using SQL Window Functions (`RANK()`).

### High Value Customers

Implemented using Common Table Expressions (CTEs).

## Docker Setup

Start PostgreSQL:

```bash
docker compose up -d
```

Connect to PostgreSQL:

```bash
docker exec -it ecommerce-postgres psql -U postgres -d ecommerce
```
